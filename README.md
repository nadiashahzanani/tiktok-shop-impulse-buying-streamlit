# Project Scientific Visualization

## Project Overview
This project explores **students' impulse buying behavior on TikTok Shop** using scientific visualization techniques. The goal is to identify key factors influencing impulse buying and present them interactively using Streamlit.

## Dataset
The dataset consists of survey responses from students about their experience and behavior on TikTok Shop. It has been cleaned and preprocessed to calculate factor scores including **Trust Score**, **Motivation Score**, and overall impulse buying score.

## Each Focuses
- **Sub-Objective 3:** Examine the role of **trust and shopping motivation** on TikTok Shop. Visualizations include bar charts, box plots, radar charts, scatter plots, and heatmaps showing relationships between trust and motivation factors.

## How to Run
1. Clone the repository.
2. Install requirements: `pip install -r requirements.txt`
3. Run Streamlit app: `streamlit run app.py`

## Tools Used
- Python  
- Streamlit  
- Pandas  
- Plotly  
- Matplotlib  
- Seaborn  
- Openpyxl

## Deployment
The application is deployed on Streamlit Cloud, providing interactive analysis of key factors influencing impulse buying behavior.
